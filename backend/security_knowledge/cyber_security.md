### Password Security
- Use unique passwords for each account
- Minimum 12 characters with mix of character types
- Use password manager for secure storage
- Enable two-factor authentication everywhere possible

### Phishing Protection
**Recognizing Phishing Attempts:**
- Urgent or threatening language
- Requests for personal information
- Suspicious sender email addresses
- Poor spelling and grammar
- Mismatched URLs

**Response to Phishing:**
- Do not click links or download attachments
- Report to IT security or abuse department
- Delete the message
- Monitor accounts for suspicious activity

### Credit Card Security
- Monitor statements weekly for unauthorized charges
- Use virtual card numbers for online purchases
- Enable transaction alerts
- Shred documents with financial information

### Identity Theft Response
**Immediate Steps:**
1. Place fraud alert on credit reports
2. Freeze credit with all three bureaus
3. File report with Federal Trade Commission
4. Contact financial institutions
